# Scientific Calculator

A simple Scientific Calculator built with Python. It supports basic arithmetic operations as well as several scientific functions like square root, trigonometry, logarithms, factorials, and mathematical constants.

## Features

- Basic Arithmetic Operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)

- Scientific Functions
  - Square Root (`sqrt()`)
  - Sine (`sin()`)
  - Cosine (`cos()`)
  - Tangent (`tan()`)
  - Log Base 10 (`log()`)
  - Natural Log (`ln()`)
  - Factorial (`factorial()`)

- Mathematical Constants
  - Pi (`pi`)
  - Euler's Number (`e`)

- Error Handling
  - Division by Zero
  - Invalid Expressions

- Power Operator Support
  - Use `^` for exponents

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/scientific-calculator.git
```

Move into the project folder:

```bash
cd scientific-calculator
```

Run the program:

```bash
python calculator.py
```

---

## Usage Examples

### Square Root

```python
sqrt(64)
```

Output:

```text
Result = 8
```

### Trigonometry

```python
sin(pi/2)
```

Output:

```text
Result = 1
```

### Factorial

```python
factorial(6)
```

Output:

```text
Result = 720
```

### Logarithm

```python
log(100)
```

Output:

```text
Result = 2
```
